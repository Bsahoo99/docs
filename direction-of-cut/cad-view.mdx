CAD View is the default tab when you first arrive at Direction of Cut. It shows the six directions of your part as clickable buttons in a grid, along with an **Analytics** panel that summarizes what is accessible from each direction.

![CAD View with Top selected and analytics panel values](/images/onboarding/screenshots/s22-direction-cad-view-top-analytics.webp)

## Six face buttons

The six buttons are: **Top**, **Bottom**, **Front**, **Back**, **Left**, **Right**

Each button has a **small number badge** in the corner. That number tells you how many features are assigned to that face direction. If the badge says 0 or is missing, no features are currently going to be machined from that direction.

**What happens when you click a face button:**
1. The button highlights in blue
2. The 3D model rotates so that face is facing you
3. That face highlights in blue on the model
4. The **Analytics** panel updates to show only what is on that face

## Analytics panel

When you click a face button, the Analytics section on the right shows a breakdown of that face:

| Analytics Field | What It Means |
|----------------|---------------|
| **Features** | Total number of features accessible from this direction |
| Feature subtypes (e.g., Slots, Independent Planes) | How many of each type are in this group — click a type to expand and see individual features |
| **Secondary Accessible Features** | Features that can also be reached from this direction as a secondary approach — not the primary direction |

## Navigating the 3D Model

| Action | How to Do It |
|--------|-------------|
| **Rotate the part** | Click and hold on the grey background (not on the part), then drag |
| **Zoom in** | Scroll the mouse wheel forward (toward the screen) |
| **Zoom out** | Scroll the mouse wheel backward (away from the screen) |
| **Pan (slide the view)** | Use toolbar tool #2, then click and drag |
| **Reset to default view** | Click toolbar tool #4 |
