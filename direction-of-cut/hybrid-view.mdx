Hybrid view is the default view in Direction of cut. It shows your 3D model alongside the feature cards, so you can see a feature on the part and review its details at the same time.

![Hybrid view with the 3D model beside feature cards grouped by direction](/images/onboarding/screenshots/s72-direction-hybrid-view.webp)

## Face buttons

The six machining directions — **Top, Bottom, Front, Back, Left, Right** — appear as buttons. Each button has a small number badge showing how many features are assigned to that direction. A badge of 0, or no badge, means nothing is machined from that direction.

Click a face button to focus that direction: the model turns to that face, the face highlights, and the view focuses on the features assigned there.

## Analytics

Alongside the model, an analytics panel summarizes the selected direction — the total number of features accessible from it, a breakdown by feature type, and any features that can also be reached from this direction as a secondary approach.

## Feature cards grouped by direction

Features are grouped by their assigned machining direction. Each group shows the direction name and a count, for example "Top — 3 features". Expand a group to see the individual [feature cards](/direction-of-cut/feature-cards) inside.

Use the grouping to check that every feature is in the right place — a hole on the top face should sit under **Top**, not **Front**. If one is in the wrong group, [change its direction](/direction-of-cut/changing-setup-direction).

<Tip>
If only two groups have features — for example only Top and Bottom — your part likely needs just two setups: one clamping for the top, one flip for the bottom. Useful to know before the Clamping strategy step.
</Tip>

## Navigate the 3D model

| Action | How to Do It |
|--------|-------------|
| **Rotate the part** | Click and hold on the grey background (not on the part), then drag |
| **Zoom in / out** | Scroll the mouse wheel forward or backward |
| **Pan (slide the view)** | Use toolbar tool #2, then click and drag |
| **Reset to default view** | Click toolbar tool #4 |
