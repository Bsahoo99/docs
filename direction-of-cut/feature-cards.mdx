Whether you are in CAD View, Setup View, or Grid View, individual features are shown as **feature cards**. A feature card is a small panel showing all the details for one feature.

![Feature card fully annotated with direction dimensions operations and flag](/images/onboarding/screenshots/s25-direction-feature-card-annotated.webp)

## Parts of a Feature Card

| Part of the Card | What It Is |
|-----------------|------------|
| **Type badge** (top-left, colored label) | The feature type — see the table below |
| **Direction badge** (e.g., "TOP") | Which setup direction is assigned to this feature. Color-coded. |
| **Flag icon** (top-right corner) | Click to flag this feature as wrong. See [Flagging Features](/direction-of-cut/flagging-features). |
| **Dimension fields** | Measurements — varies by type. Independent Plane shows Area. Through Hole shows Radius and Depth. |
| **Operations dropdown** | Shows "Operations X/Y" — click the arrow to expand and see the full list of operations for this feature. |

## Feature Types Explained

| Feature Type | What It Means in the Shop |
|-------------|---------------------------|
| **Independent Plane** | A flat machined surface — a face that needs to be faced or milled flat |
| **Through Hole** | A hole that goes all the way through the part |
| **Counterbore** | A stepped hole — wider at the top, then narrower going deeper. Used for socket head cap screws so the head sits flush. |
| **Slot** | A channel cut into the part |
| **Fillet** | A curved blend between two surfaces — an internal corner radius or external rounded edge |
| **Curved Surface** | A free-form surface that cannot be described as flat. Requires a ball end mill. |
| **OD Profile** | The outer diameter profile — for parts with a stepped or contoured outer shape |
| **3D Surface** | A complex sculptured surface that needs 3D surfacing toolpaths |

## Direction Color Codes

| Direction | Badge Color |
|-----------|------------|
| Top | Blue |
| Bottom | Purple |
| Front | Green |
| Back | Orange / Red |
| Left | Pink / Rose |
| Right | Yellow |
