Each strategy represents one physical clamping setup — one time the part is in the workholding.

## Open the New Strategy form

On the Clamping strategy screen, click **+ Add Clamping Strategy**. A **New Strategy** form opens as a dashed-border card.

![New strategy form with the face grid, workholding, and cycle time parameters](/images/onboarding/screenshots/s37-clamping-new-strategy-form.png)

## Set the datum and blocked faces

The form opens with an interactive **6-face grid**. This is where you tell Neuramill how the part sits in the workholding and which faces the tool cannot reach.

- **Click a face** to set it as the **datum** — the face sitting down on the workholding. It turns red.
- **Right-click** (or **Shift+click**) a non-datum face to mark it **blocked** — a face the workholding jaw or clamp covers. It turns orange. Do it again to unblock.
- Any face that is neither datum nor blocked is **accessible** — the tool can reach it in this setup.

| Color | Meaning |
|-------|---------|
| **Red** | Datum — the part sits on this face |
| **Orange** | Blocked — the workholding covers this face; the tool cannot reach it |
| **Green** | Accessible — the tool can reach this face in this setup |

![Face grid showing datum (red), blocked (orange), and accessible (green) faces](/images/onboarding/screenshots/s38-clamping-face-buttons-color-states.png)

<Tip>
Setting the datum and blocked faces correctly is the most important part of the strategy — any face you leave accessible is one the tool will try to machine in this setup.
</Tip>

## Choose a workholding

Open the **Workholding** dropdown and select the device you are using from your company's [workholding library](/dashboard/machining-config). If the dropdown is empty, no workholdings have been set up yet.

Selecting a workholding fills in its default **Load / Unload** and **Setup Datum** times automatically — you can adjust them below.

## Set cycle time parameters

The **Cycle Time Parameters** block sets the non-cutting time for this setup, in seconds. These feed the [cycle time estimate](/summary/cycle-time) on the Summary step.

| Parameter | What it covers |
|-----------|----------------|
| **Probe Cycle Time** | Probing the part to confirm its position. Always editable. |
| **Load / Unload Time** | Loading the part into the workholding and removing it. Enabled once you select a workholding. |
| **Setup Datum Time** | Setting the work origin for this setup. Enabled once you select a workholding. |

Each value must be 0 or greater. You cannot add the strategy while a field is blank or negative.

## Add notes (optional)

Type any reminders in the **Notes** field — for example *"6-inch Kurt vise, 0.5-inch step jaws."* Notes appear on the strategy card and in the Summary.

## Add the strategy

Click **Add Strategy**. The form closes and a **strategy card** appears.
