At the top of the Clamping Strategy panel, four counters keep track of your progress.

![Feature distribution tracker with Total Claimed Remaining and Ready counters](/images/onboarding/screenshots/s36-clamping-feature-distribution-tracker.webp)

## Distribution counters

| Counter | What It Shows |
|---------|---------------|
| **Total** | The total number of features to be machined |
| **Claimed** | Features that have been assigned to at least one approved strategy |
| **Remaining** | Features not yet covered by any strategy. **Your goal is zero.** |
| **Ready** | Features fully assigned and confirmed |

## Using the remaining counter

Click the **Remaining** counter number to highlight all uncovered features in blue on the 3D model. This shows you exactly which faces still need a clamping strategy.

Use this at the end after adding each strategy — it is the fastest way to see what is still left to cover.

<Warning>
If you approve your clamping step with Remaining > 0, those features will not appear in tool recommendations or your final machining plan. Always check this counter before moving forward.
</Warning>
