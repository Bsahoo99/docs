---
title: "Filter tool sources"
description: "Control which tool library the ITR searches — your company's tools, the Neuramill catalog, or specific vendors."
---
Before or after selecting tools, you can control which pool of tools Neuramill searches. Click the **filter icon** (funnel icon) near the Select Optimal Tools button.

## Source options

| Source | What It Searches |
|--------|-----------------|
| **Company Tools** (checked by default) | Only tools your company already owns. These come from your tool library, loaded by the Neuramill team. On Trial plans, falls back to the Neuramill general catalog. |
| **Neuramill Catalog** (unchecked by default) | All tools from Neuramill's full vendor database — including tools you do not own yet. Useful for finding tools to order. |

## Vendor checkboxes

Below the source options, you will see checkboxes for individual **vendors** (tool suppliers). Examples: Haas Automation, IMCO Carbide, Bits & Bits.

- **Check** a vendor to include their tools in the search
- **Uncheck** a vendor to exclude their tools

This lets you narrow the search to brands you prefer to use in your shop.

## When to use the Neuramill catalog

<Tip>
If the system shows **"No match found"** for an operation using Company Tools only, enable the Neuramill Catalog. This shows you tools that would work — tools you could order and add to your library. Screenshot the recommendation and pass it to your purchasing department.
</Tip>

## Operation and feature filters

At the top of the ITR results area, two filter dropdowns help you navigate when you have many operations:

| Filter | What It Does |
|--------|-------------|
| **Operation filter** | Shows tools for one operation type only: All, FACING, ROUGHING, FINISHING, SPOT DRILLING, DRILLING |
| **Feature filter** | Shows tools for one specific feature only |
