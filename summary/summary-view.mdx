The Summary is the fifth and final tab in the session navigation bar. It shows a complete read-only review of everything configured in your session.

## Get to the summary

Click the **Summary** tab in the navigation bar at the top of the screen. The tabs read:

> Machining Setup → Direction of Cut → Clamping Strategy → Intelligent Tool Recommender → **Summary**

<Note>
The Summary tab is greyed out until the Intelligent Tool Recommender step has been approved. If the tab is unresponsive, go back and confirm that Step 4 shows a green "Tools Selected" badge.
</Note>

## Summary details

The Summary is **view-only**. You cannot edit anything directly on this screen. It is organized into two sections.

### Top section — setup overview

Three panels side by side:

| Panel | What It Contains |
|-------|-----------------|
| **Machine** | Machine name, description, max speed, max torque, max rating, taper, max cutting feed |
| **Material** | Material name, hardness range, yield strength, tensile strength |
| **Stock & Rigidity** | Stock dimensions (X × Y × Z) and the selected rigidity level |

### Lower section — strategies

Each clamping strategy appears as a card showing:
- The 6-face grid (red = datum, orange = blocked, green = accessible)
- Workholding name and notes you entered
- Every operation pass with: operation type, number of features, policy badge, and the selected tool card with all cutting parameters

![Summary view with the setup panels and a strategy card with tool cards](/images/onboarding/screenshots/s56-summary-view-full.webp)

## Read a summary operation row

Each row in the strategy shows:

| Element | What It Is |
|---------|-----------|
| Operation name | FACING, ROUGHING, FINISHING, DRILLING, etc. |
| Feature count | How many features are in this pass |
| Policy badge | Whether this uses the Default policy or a Custom policy, and which tier (Ideal / Moderate / Worst) |
| Tool name | The selected tool |
| Diameter | Tool diameter |
| RPM | Spindle speed |
| Feed | Cutting feedrate |
| Plunge | Plunge feedrate |

## Estimate a price

The Summary step also has a **Quote** button that opens [Quick Quote](/summary/quote-calculator), a preview tool that estimates a price from the plan's cycle time and your shop rates.
