The Machines tab shows every CNC machine that has been set up for your company in Neuramill. These are the machines you can select when you reach the Machining Setup step of a session.

<Note>
Machines are added by Admins and Super Admins only. If you are a regular User, you can view this tab but cannot add or remove machines. If a machine is missing, ask your Admin.
</Note>

## Filter the machine list

Click the **Filters** button to narrow down the list by machine capabilities.

![Machines tab with RPM rating and cutting speed filters open](/images/onboarding/screenshots/s08-dashboard-machines-filters-open.webp)

| Filter | What It Does |
|--------|-------------|
| **RPM range** | Shows only machines whose maximum spindle speed falls within the range you set. Drag the slider handles to set a minimum and maximum RPM. |
| **Rating range** | Filters by maximum motor power (kW). |
| **Cutting speed range** | Filters by maximum cutting feed rate (m/min). |

## Machine details

Click on any machine to see its full specifications.

| Spec | Example (UMC-500) | What It Means |
|------|-------------------|---------------|
| Description | 5-Axis CNC Vertical Machine | Brief description of the machine |
| Max Speed | 8100 rpm | Highest spindle speed available |
| Max Torque | 122.0 Nm @ 2000 rpm | Maximum twisting force of the spindle |
| Max Rating | 22.4 kW | Motor power |
| Taper | CT40 \| BT40 \| HSK-A63 | Spindle taper — determines which tool holders fit this machine |
| Max Cutting Feed | 16.5 m/min | Fastest table movement speed |
