The **Machining configuration** area is the hub for the libraries that feed planning: materials, machines, workholdings, and the tool catalog. From here you can view and manage these libraries.

<Info>
Any team member can add to and edit these libraries. The one exception is **adding machines**, which is restricted to admins.
</Info>

![Machining configuration area showing the library sections](/images/onboarding/screenshots/s69-dashboard-machining-config.webp)

## Libraries

| Library | What it holds |
|---------|---------------|
| Materials | The stock materials Neuramill plans against, with their machining characteristics. |
| Machines | The CNC machines configured for your company. |
| Workholdings | The clamping strategies available when you plan a setup. |
| Tool catalog | Your company's cutting tool library. |
