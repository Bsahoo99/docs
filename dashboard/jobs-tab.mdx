The Jobs tab is the first thing you see when you log in. It lists every machining session your team has created.

{/* ![Jobs tab showing the session table with all columns](/images/onboarding/screenshots/s06-dashboard-jobs-table.webp) */}

## Jobs table columns

| Column | What It Shows |
|--------|---------------|
| **Job ID** | A unique number given to each session automatically (e.g., #76). Write this down if you ever need to report a problem — the support team will ask for it. |
| **Machine** | Which CNC machine was selected for this job. Blank if not yet set. |
| **Material** | The workpiece material (e.g., Aluminium 6061). Blank if not yet set. |
| **Stock Dimensions** | The X × Y × Z size of the raw stock block (e.g., 305 × 254 × 73 mm). |
| **Features** | How many manufacturing features were found in the CAD file. |
| **Operations** | How many machining operations the plan includes. |
| **Rating** | A star rating (1–5) you give the job after it is complete. Blank if not yet rated. |
| **Status** | Whether the job is In Progress or complete. |
| **User** | The name of the team member who created this session. |

## Open an existing session

To reopen a job you have already started, **click anywhere on that job's row**. The session opens where you left off.

## Use filters

The **Filters** button is near the top of the Jobs tab. Click it to open the filter panel and narrow down the list.

![Jobs tab with filters panel open and filter controls visible](/images/onboarding/screenshots/s07-dashboard-jobs-filters-open.webp)

| Filter | How to Use It |
|--------|--------------|
| **Filter by Machine** | Click the dropdown and choose a machine name. Only jobs that used that machine are shown. |
| **Filter by Material** | Click the dropdown and choose a material. Only jobs with that material are shown. |
| **Filter by User** | Click the dropdown and choose a team member's name. Only their jobs are shown. |
| **Features (range slider)** | Drag the left and right handles to set a minimum and maximum feature count. Example: set to 10–50 to see only jobs with 10 to 50 features. |
| **Operations (range slider)** | Same idea — filter by number of operations. |
| **Ratings (range slider)** | Filter by the star rating you gave completed jobs. |

To reset all filters and show everything again, look for the **Clear Filters** or **Reset** button at the bottom of the filter panel.
