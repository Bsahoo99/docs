The Tools tab shows every cutting tool in your company's tool library. This is the library that Neuramill searches through when it recommends tools for your jobs.

<Warning>
Tools are loaded and managed by the **Neuramill team** during your company onboarding. You cannot add or remove tools yourself. If a tool is missing or incorrect, contact the Neuramill support team.
</Warning>

## Tool entry details

| Field | What It Means |
|-------|---------------|
| Tool ID | A unique number for this tool in the system |
| Name | The tool's product name and catalog number |
| Type | The category: Flat End Mill, Ball End Mill, Drill, Spot Drill, Reamer, etc. |
| Diameter | The cutting diameter in mm |
| Flutes | The number of cutting edges |
| Cutting Length | How deep the tool can cut in one pass |
| Corner Radius | The radius on the tool tip (relevant for end mills) |
| Vendor | The manufacturer or supplier |

## Filter the tool list

Use the filter panel to search your tool library by type, vendor, diameter range, flute count, cutting length, or corner radius.
