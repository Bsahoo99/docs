The **Machine** dropdown is in the top-right section of the Machining Setup panel.

## Select your machine

<Steps>
  <Step title="Click the Machine dropdown">
    Click the dropdown field labeled **MACHINE**. Your company's available machines are listed. These were configured by your Admin or the Neuramill team.
  </Step>
  <Step title="Click the machine you plan to use">
    Click the machine name that matches the CNC machine you will be running this job on.
    A **Machine Specs** panel expands below.
    
    ![Machine dropdown selected with expanded machine specs](/images/onboarding/screenshots/s32-machining-machine-dropdown-specs.webp)
  </Step>
</Steps>

## Machine specs explained

| Spec | Example (UMC-500) | What It Means |
|------|-------------------|---------------|
| Description | 5-Axis CNC Vertical Machine | Brief description |
| **Max Speed** | 8100 rpm | The highest spindle speed on this machine |
| **Max Torque** | 122.0 Nm @ 2000 rpm | Maximum twisting force of the spindle |
| **Max Rating** | 22.4 kW | Motor power |
| **Taper** | CT40 \| BT40 \| HSK-A63 | Spindle taper type — determines which tool holders fit this machine |
| **Max Cutting Feed** | 16.5 m/min | Fastest table feed rate |

<Note>
If the machine you need is not in the list, contact your Admin. Only Admins and Super Admins can add machines to the system — regular Users cannot.
</Note>
