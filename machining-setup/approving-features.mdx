title: "Approve extracted features"
description: "Before setting material and machine, you approve the final feature list to lock it in."
---
At the top of the Machining Setup panel, there is an **Extracted Features** section. This shows the final list of features that will be carried through the rest of the session.

## Approve

1. Look at the feature count (e.g., "Extracted Features (5)").
2. Scroll through the feature cards one more time to confirm everything looks right.
3. When satisfied, click the **Approve Features** button.
4. The button changes to a green **✓ Approved** badge. The feature list is now locked for this session.

## If you need to change something

The detailed review — confirming each feature's machining direction and operations, or flagging features the AI got wrong — happens in the next step, **Step 2 — Direction of cut**. Approving the extracted features here just confirms the list Neuramill detected.

1. Approve the features to unlock [Step 2 — Direction of cut](/direction-of-cut/overview).
2. Review and correct individual features there.
3. If you later return to Machining Setup, your material, machine, and stock settings are preserved.

<Note>
Moving between steps does not erase the settings you have already entered. They are still there when you return.
</Note>
